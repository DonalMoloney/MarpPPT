## Write Slides In Markdown

- Keep text short and focused.
- Put one main idea on each slide.
- Store reusable images in `assets/`.
- Store generated charts in `figures/`.

## Review And Export

- Preview while editing: `npx marp slides.md --preview`
- Export to PowerPoint: `npx marp slides.md --pptx`
- Check the Notes pane before sharing.
